# Capstone

## Project Details

-   Truffle v5.1.49 (core: 5.1.49)
-   Solidity v0.5.16 (solc-js)
-   Node v14.9.0
-   Web3.js v1.2.1

## Running Tests Instructions

-   npm install
-   cd eth-contracts
-   ganache-cli
-   truffle migrate --reset
-   truffle test
